# ROBOT Browser V9 — Market Maker Detection Rules

## Цель

ROBOT Browser V9 ищет не новые токены, не дешёвые монеты и не максимальный объём.

Главная цель — найти рынки, которые выглядят управляемыми алгоритмом маркет-мейкера и подходят для наблюдения или скальпинга.

Идеальный рынок выглядит так:

- сделки проходят по реальным уровням стакана;
- размеры сделок повторяются;
- цена движется плавно;
- стакан достаточно плотный;
- нет хаотичных рывков;
- рынок выглядит управляемым одним алгоритмом.

---

## 1. Execution Ratio

Execution Ratio — главный признак.

Он показывает, насколько сделки соответствуют видимым уровням стакана.

### Хорошие значения

| Значение | Оценка |
|---:|---|
| 90–100 | Отлично |
| 75–89 | Хорошо |
| 60–74 | Средне |
| ниже 60 | Плохо |

Если Execution ниже 60, рынок не должен получать высокий итоговый Score.

---

## 2. Uniformity

Uniformity показывает повторяемость размеров сделок.

Если сделки часто проходят одинаковыми или почти одинаковыми объёмами, это сильный признак алгоритма.

### Хорошие значения

| Значение | Оценка |
|---:|---|
| 85–100 | Отлично |
| 70–84 | Хорошо |
| 50–69 | Средне |
| ниже 50 | Плохо |

Если Uniformity ниже 40, Confidence должен быть LOW.

---

## 3. Price Smoothness

Price Smoothness показывает, насколько плавно движется цена.

Нужны рынки, где цена словно "ведётся", а не прыгает хаотично.

### Хорошие значения

| Значение | Оценка |
|---:|---|
| 85–100 | Отлично |
| 70–84 | Хорошо |
| 50–69 | Средне |
| ниже 50 | Плохо |

---

## 4. Spread

Spread не является главным фильтром.

Он отображается для оценки удобства скальпинга.

### Ориентиры

| Spread | Оценка |
|---:|---|
| до 0.05% | Отлично |
| 0.05–0.20% | Нормально |
| выше 0.20% | Осторожно |

---

## 5. Depth

Depth показывает плотность стакана в Top5 Bid и Top5 Ask.

Используются значения:

- Top5 Bid
- Top5 Ask
- Top5 Total

Depth пока не является главным фильтром, но будет использоваться в будущей версии Score.

---
