# ROBOT Browser V9 — Metrics Registry

## Цель

Все вычисляемые признаки рынка являются независимыми метриками.

Каждая метрика:

- вычисляется отдельно;
- ничего не знает о других метриках;
- возвращает одно значение;
- имеет единый интерфейс.

---

## Интерфейс

Все функции должны иметь одинаковую сигнатуру.

calc_metric(
    trades,
    orderbook,
